# Products & Services

## KYC Verification
**Starting at 0.25 ETH**

### Process
1. Submit KYC request via online form
2. Access secure dashboard for document uploads
3. Complete live video verification call with specialist
4. Receive official KYC certification and minted NFT

### Verification Components
- Document scanning and validation
- Biometric analysis
- Device analytics
- Liveness checks (anti-spoofing)
- Sanctions screening
- Mandatory video interview

### Deliverables
- Official KYC certification
- Custom NFT minted with project name and logo
- Project listing in verified registry
- Verification badge for website/socials

### Timeline
- Standard turnaround: 24-48 hours from payment and form completion
- Includes custom content generation

### Requirements
- Minimum one key leadership member with contract, treasury, or liquidity control must verify
- Full team verification encouraged but optional

## Smart Contract Audits
**Starting at 0.3 ETH**

### Timeline
- Standard completion: 48 hours from order
- Expedited 24-hour options available

### Deliverables
- Detailed audit report
- Audit certificate
- Dedicated project page
- Website badge
- Block explorer integration (Etherscan, etc.)

### Revision Policy
- Standard audits: 1 free revision
- Advanced audits: 2 free revisions
- Additional revisions available for fee

### Audit Types
- Standard smart contract audit
- Advanced audit (consensus, networking, node implementation)
- Custom contract development available
