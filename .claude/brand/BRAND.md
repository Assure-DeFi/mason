# Assure DeFi® Brand Rules (Authoritative)

This folder defines the visual and UX rules for this project.
All UI and front-end work must comply with this file.

## Brand Pillars
- Integrity
- Trust
- Credibility

## Brand Ethos
Trust, but verify.

## Approved Colors ONLY
- Navy: #0A0724
- Gold: #E2D243
- Light Grey: #F2F2F2
- White: #FFFFFF
- Black: #000000

No other colors may be introduced.

## Typography
Primary: Akzidenz Grotesk (if available)
Fallback: Inter

- Headings: Inter 600–700
- Body: Inter 400
- UI labels: Inter 500

## UI Tone
- Dark-mode first
- Serious, professional, high-trust
- No playful or consumer-style UI

## Logo Usage
- Use only assets in /brand/assets/logos
- Do not recolor, stretch, rotate, or add effects
- Maintain clear space around logos
