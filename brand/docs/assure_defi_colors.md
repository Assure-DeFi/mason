# Assure DeFi Official Brand Colors

## Extracted from Official Logo Pack

### Primary Brand Colors:

1. **Gold** (Primary Accent)
   - Hex: `#E2D243`
   - RGB: rgb(226, 210, 67)
   - Usage: Primary accent color, CTAs, highlights, logo accents

2. **Navy** (Primary Dark)
   - Hex: `#0A0724`
   - RGB: rgb(10, 7, 36)
   - Usage: Primary background, dark elements, text on light backgrounds

3. **White** (Primary Light)
   - Hex: `#FFFFFF`
   - RGB: rgb(255, 255, 255)
   - Usage: Text on dark backgrounds, contrast elements

4. **Black** (Alternative Dark)
   - Hex: `#000000`
   - RGB: rgb(0, 0, 0)
   - Usage: Alternative dark option for maximum contrast

## Color Palette for Mason Character

The Mason character should incorporate all these colors:
- **Primary**: Navy (#0A0724) for the hooded figure and shield
- **Accent**: Gold (#E2D243) for the glowing elements, eyes, and highlights
- **Contrast**: White (#FFFFFF) for the book pages and light elements
- **Background**: Can use Navy or Black depending on context

## Brand Characteristics:
- Premium, professional aesthetic
- Security and trust-focused
- High contrast design philosophy
- Geometric, clean shapes
- Mystical/magical elements (star, glow effects)
