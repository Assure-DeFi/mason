# Mason Brand Kit

Official brand assets for Mason - the AI-powered software development assistant.

## Quick Start

### Folder Structure

```
brand/
├── logos/
│   ├── svg/           # Vector logos (use for web, print)
│   └── png/           # Raster logos (use for presentations)
├── favicons/          # Complete favicon package
├── docs/              # Brand documentation
└── showcase/          # Brand showcase images
```

### Which Logo to Use

| Use Case            | Recommended File                         |
| ------------------- | ---------------------------------------- |
| Website header      | `logos/svg/mason_logo_horizontal.svg`    |
| App icon            | `logos/svg/mason_icon.svg`               |
| Marketing materials | `logos/svg/mason_logo_primary.svg`       |
| Dark backgrounds    | `logos/svg/mason_logo_primary_white.svg` |
| High-res print      | `logos/png/mason_logo_v1_primary.png`    |
| Favicon source      | `logos/png/mason_logo_v5_minimal.png`    |

## Brand Colors

| Color     | Hex       | Usage                           |
| --------- | --------- | ------------------------------- |
| **Navy**  | `#0A0724` | Primary background, figure fill |
| **Gold**  | `#E2D243` | Accents, shield frame, star     |
| **White** | `#FFFFFF` | Text on dark, face opening      |

## Favicon Usage

Copy the contents of `favicons/` to your public directory and add to your HTML:

```html
<link rel="icon" href="/favicon.ico" sizes="any" />
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png" />
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png" />
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
<link rel="manifest" href="/site.webmanifest" />
<meta name="theme-color" content="#0A0724" />
```

## Typography

- **Wordmark font:** Poppins Bold
- **Letter spacing:** 0.1em
- **Fallbacks:** Inter, Segoe UI, Arial

## Guidelines

### Do

- Use provided logo files without modification
- Maintain clear space around logos
- Use appropriate color variants for backgrounds
- Scale logos proportionally

### Don't

- Change logo colors
- Stretch or distort logos
- Add effects (shadows, glows, gradients)
- Rotate the logo
- Place on low-contrast backgrounds
